# 登戸タイムズ 編集ガイドライン

記事を書くとき（人間・AIとも）は必ずこのルールに従うこと。

## 大原則: 事実のみを書く

- **Web検索・公式発表で確認できた事実だけを書く。憶測・創作・記憶頼みの記述は禁止。**
- すべての重要な事実（日付、店名、価格、施設規模など）は出典URLで裏付け、frontmatterの `sources` に記載する。
- 一次情報（公式サイト、プレスリリース、自治体発表）を優先する。まとめサイトのみが根拠の場合は「報道によると」「〜とのことです」と伝聞であることを明示する。
- 情報源同士で数字が食い違う場合は、より信頼できる方を採用し、断定を避ける表現にする。
- **日付の相対性に注意**: 「2026年春オープン予定」という古い記事を見つけても、現在すでにその時期を過ぎていれば「開業済みか」を必ず追加検索で確認する。予定の話と実績の話を混同しない。

## 文体・構成

- です・ます調。地域住民に語りかける親しみやすいトーンで、煽りや誇張はしない。
- 本文400〜800字程度。導入（何が起きたか）→ 見出し付きの詳細 → 地域への影響や周辺情報、の構成。
- 見出しは `##` を使う。1記事に1〜3個程度。
- タイトルには「登戸」「向ヶ丘遊園」など対象の地名を含める（ローカルSEO対策）。
- 日付は「2026年3月25日（水）」のように和暦なし・曜日ありが望ましい。

## やってはいけないこと

- 画像の無断転載（当面、記事に画像は使わない）
- 店舗・個人への批判、未確認の閉店・不祥事情報の掲載
- 口コミサイトの評価点数の転載（「人気です」程度の表現に留める）
- 既存記事と同じネタの再掲（`data/coverage-log.json` と `src/content/posts/` を必ず確認）

## 対象エリア

登戸駅・向ヶ丘遊園駅の周辺（川崎市多摩区）を中心に、宿河原・中野島など隣接エリアも可。それより遠い話題は、登戸住民への影響が明確な場合のみ（例: 小田急線・南武線のダイヤ改正）。

## ネタがない日の運用

- 新しいニュースがなければ無理に書かず、`data/coverage-log.json` の `skips` に日付と確認した検索クエリを記録して終了する。
- ただし直近3日以上新しい記事が出ていない場合は、「登戸の人気ランチまとめ」のような既存店舗のまとめ記事（カテゴリ: gourmet）を1本作成する。まとめ記事も全店舗に出典が必要。
